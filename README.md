# Zomato Data Analysis Using Python

## Overview
This project analyzes Zomato restaurant data using Python libraries like Pandas, NumPy, Matplotlib, and Seaborn.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Ratings Analysis
- Online vs Offline Orders
- Restaurant Type Analysis
- Cost Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains restaurant information such as:
- Restaurant type
- Votes
- Ratings
- Online order availability
- Approximate cost for two people

## Key Insights
- Most restaurants belong to the dining category
- Online-order-enabled restaurants received higher ratings
- Couples preferred restaurants costing around ₹300
- Dining restaurants mainly support offline orders
