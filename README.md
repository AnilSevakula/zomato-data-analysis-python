# Zomato Data Analysis Using Python

## Project Overview
This project analyzes Zomato restaurant data using Python to uncover customer preferences, restaurant trends, and business insights.

## Objectives
- Compare online vs offline order preferences
- Identify popular restaurant categories
- Analyze restaurant ratings
- Determine preferred spending range for couples
- Visualize restaurant trends using charts and heatmaps

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains restaurant information such as:
- Restaurant type
- Votes
- Ratings
- Online order availability
- Approximate cost for two people

## Key Insights
- Most restaurants belong to the dining category
- Online-order-enabled restaurants received higher average ratings
- Couples preferred restaurants with an approximate cost of ₹300
- Dining restaurants mainly support offline orders

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Business Insights Generation

## Visualizations
- Restaurant Type Count Plot
- Ratings Distribution Histogram
- Online vs Offline Orders
- Heatmap Analysis
- Rating Comparison Boxplot

2. Install required libraries:


